# Task Manager - Мобильное приложение

## Описание
Приложение для управления задачами с фильтрацией по статусам. Разработано в рамках учебного проекта по ТРПО.

## Функционал
- ✅ Создание и отображение задач
- ✅ Фильтрация задач по статусам (Все, Сделать, В процессе, Завершено)
- ✅ Добавление новых задач
- ✅ Карточки задач с заголовком, описанием, датой и статусом

## Технологии
- **Язык:** Kotlin
- **Минимальная версия Android:** 7.0 (API 24)
- **Архитектура:** MVVM (начальная реализация)
- **Библиотеки:** AndroidX, Material Design
